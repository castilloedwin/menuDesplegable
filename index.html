Menú Desplegable v. 1.0

Es un menú desplegable, dando al usuario justo lo que necesita, que es poder tener acceso a múltiples opciones que brinda la Web. Su sencillez permite al usuario tener una experiencia de navegación bastante cómoda. A continuación veremos como implementarlo:

Estructura HTML

<header>
   <div id="contentHeader">
      <ul>
         <li><a href="#">Inicio</a></li>
         <li class="q"><a href="#">Quiénes Somos</a>
            <ul>
               <li><a href="#">Nuestra Misión</a></li>
               <li><a href="#">Nuestra Visión</a></li>
            </ul>
         </li>
         <li class="q"><a href="#">Portafolio</a>
            <ul>
               <li><a href="#">Diseño Gráfico</a></li>
               <li><a href="#">Diseño Web</a></li>
               <li class="r"><a href="#">Marketing Digital</a>
                  <ul>
                     <li><a href="#">Facebook</a></li>
                     <li><a href="#">Twitter</a></li>
                     <li><a href="#">LinkedIn</a></li>
                  </ul>
               </li>
           </ul>
         </li>
         <li><a href="#">Historia</a></li>
         <li><a href="#">Contáctenos</a></li>
      </ul>
   </div>
</header>
